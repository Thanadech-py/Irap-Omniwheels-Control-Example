<h1>Omnidirectional Wheels Control</h1>

The Omnidiectional Wheels are mostly use in robotics because it made the robot has abilities to move
in every direction possible, we called "**Omnidirectional**".

---

The program in this repository is the example of how we can control the **Omnidirectional Wheels**. 
So if yout wanted to use this repository please just read and study the concept the program haven't
made for every robot, So this maybe have a conflict with your project.
